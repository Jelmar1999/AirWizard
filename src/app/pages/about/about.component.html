Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

AirWizard

Deze website zal als functionaliteit hebben dat piloten kunnen bekijken waar zij willen en/of kunnen landen op verschillende vliegvelden. Er is informatie op te halen over verschillende vliegvelden, de gates die deze vliegvelden hebben en de vliegtuigen die hier al eventueel staan. Het is mogelijk de details van een gate te wijzigen stel dat deze in onderhoud is of iets dergelijks. Ook kan een vliegveld nieuwe gates aanleggen.

Entiteiten

De Entiteiten die deze applicatie bevatten zijn als volgt: Een vliegveld, een gate/landingsbaan en een vliegtuig. De specifieke attributen van deze entiteiten zijn hieronder te zien. Een vliegveld heeft een aantal gates. Een gate kan maximaal een vliegtuig tegelijk ontvangen. Een gate kan ook geen vliegtuig hebben. De user entiteit zal een birthdate van het type DATE hebben en een gender die uit een ENUM gender kan worden geselecteerd. De User zal ook andere users kunnen volgen en activiteiten van deze users zien.

Airport

  • airportName: string
  • airportId: Guid
  • address: string
  • gates: Gate[]

Gate

  • gateName: string
  • gateId: Guid
  • waitingRoomCapacity: number
  • currentAirplane: Airplane
  • airport : Airport

Airplane

  • planeId: Guid
  • airplaneName: string
  • model: string
  • buildYear: Date
  • fuelCapacity: number
  • length: number
  • wingspan: number
  • heigth: number
  • engine: string
  • weightClass: weightClass (ENUM)

User

  • userId: Guid
  • firstName: string
  • lastName: string
  • emailAddress: string
  • phoneNumber: string
  • dateOfBirth: date
  • gender: gender (ENUM)
  • ownedAirplanes : Airplane[]
