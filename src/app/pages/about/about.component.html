Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

AirWizard

Deze website zal als functionaliteit hebben dat piloten kunnen bekijken waar zij willen en/of kunnen landen op verschillende vliegvelden. Er is informatie op te halen over verschillende vliegvelden, de gates die deze vliegvelden hebben en de vliegtuigen die hier al eventueel staan. Het is mogelijk de details van een gate te wijzigen stel dat deze in onderhoud is of iets dergelijks. Ook kan een vliegveld nieuwe gates aanleggen.

Entiteiten

De Entiteiten die deze applicatie bevatten zijn als volgt: Een vliegveld, een gate/landingsbaan en een vliegtuid. De specifieke attributen van deze entiteiten zijn hieronder te zien. Een vliegveld heeft een aantal gates, het aantal gates is afhankelijk van de vliegbewegingen van het vliegveld. Een gate kan maximaal een vliegtuig tegelijk ontvangen. Een gate kan ook geen vliegtuig hebben. De user entiteit zal een birthdate van het type DATE hebben en een gender die uit een ENUM gender kan worden geselecteerd.

Airport

  • airportname: string
  • airportId: number
  • address: string
  • gates: gate[]

Gate

  • gatename: string
  • gateId: number
  • waitingroomcapacity: number
  • currentairplane: Airplane

Airplane

  • planeId: number
  • airplanename: string
  • model: string
  • buildyear: Date
  • fuelcapacity: number
  • length: number
  • wingspan: number
  • heigth: number
  • engine: string
  • weight: weight (ENUM)

User

  • userId: number
  • firstname: string
  • lastname: string
  • email: string
  • phonenumber: string
  • birthdate: date
  • gender: gender (ENUM)
